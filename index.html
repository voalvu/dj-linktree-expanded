Proxy Server Simulation

This page simulates a proxy server that was originally built with Express.js. The actual server would run at http://localhost:3006, but this is a static HTML representation.

GET /cors/:url

Fetches data from a provided URL and returns it. The URL is passed as a parameter, decoded, and fetched. The response is sent as JSON or text based on the content type.

Example Request: /cors/https%3A%2F%2Fwww.example.com

This would fetch content from https://www.example.com.
Simulated Response (Text Example):
<!DOCTYPE html><html>... (HTML content from example.com)
Simulated Error Response:
{ "error": "Failed to fetch data" }

GET /soundcloud/:url

Fetches data from a SoundCloud URL provided as a parameter. The response is returned as JSON or text based on the content type.

Example Request: /soundcloud/https%3A%2F%2Fsoundcloud.com%2Fsome_artist

This would fetch content from https://soundcloud.com/some_artist.
Simulated Response (Text Example):
<!DOCTYPE html><html>... (HTML content from SoundCloud page)
Simulated Error Response:
{ "error": "Failed to fetch data" }

Note: This is a static HTML page and cannot actually fetch data or run a server. The original code requires Node.js, Express, and a runtime environment to function.
